How to Apply CSS to HTML

you can reference an external CSS stylesheet using the <link> element.
Example: <link rel="stylesheet" href="styles.css" />
href needs to reference a file in your system.

How to Use JavaScript in a Webpage

To link an external JavaScript script, use <script> with src pointing to the script file.
Example: <script src="path/to/my/script.js"></script>
You can also add JavaScript directly in the html file by putting the code between script tags.
Example: <script> window.addEventListener("load", () => { console.log("This function is executed once the page is fully loaded"); }); </script>
